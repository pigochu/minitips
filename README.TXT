2. 目前功能
   * 重複時間提示 (月/周/日/時/分/秒)
   * 特定時間提示
   * 可設定提示音效
   * 隱藏在右下角

3. 適用環境
   目前作者僅於 Windows 7 上開發測試
   採用 .NET 2.0 C# 程式撰寫 , 理論上應該可以在有安裝 .NET 2.0 環境上執行
